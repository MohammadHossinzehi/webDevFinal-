1. Install Node.js

2. Change your directory to the "webDevFinal-" directory

3. Open a terminal and use to following command: "node server.js"

4. Your browser should open with a new tab which contains our project 
   but in the event that it does not go to your preferred browser and copy and paste the local host link given by the terminal into your search bar

5. Log in using username and password as "admin" or create sign up
